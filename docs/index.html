LocalLayer

On-device text translation powered by ML Kit

📱 Jetpack Compose 🎨 Material 3 🧠 ML Kit Translate 🔒 100% Offline

🌐 30+ Languages

Translate between English, French, German, Japanese, Hindi, Arabic, and many more.

📴 Fully Offline

All translation runs on-device via ML Kit. No internet connection required after model download.

⚡ Instant Swap

Swap source and target languages instantly. Type, translate, and copy with a single tap.

📋 One-Tap Copy

Copy translated text to clipboard with haptic feedback. Share results anywhere.

📦 Auto Model Download

Language models download automatically on first use. Wi-Fi recommended (≈30MB per pair).

🎨 Material Design 3

Modern UI with dynamic color (Material You). Adapts to light and dark themes.
